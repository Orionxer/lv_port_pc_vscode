# 构建说明

## 平台信息
- **构建平台**: Linux x86_64 主机
- **运行平台**: Luckfox Aura RV1126B

## 同步drm库
> 默认烧录的Buildroot系统中的`/usr/lib/`目录中没有`libdrm.so.2`，需要手动同步更新
```sh
# 开发板查找libdrm.so.2所在路径
find / -name libdrm.so.2
=============================
# 例如返回
/oem/usr/lib/libdrm.so.2
```
```sh
# 同步libdrm.so.2到/usr/lib/
cp /oem/usr/lib/libdrm.so.2 /usr/lib/libdrm.so.2
```

## 如何编译
```bash
./auto_build.sh
```

## 可执行文件位置
编译完成后，可执行文件位于：`build/lvgl_app`

## 编译工具链
- **路径**: `/sdk/tools/linux/toolchain/aarch64-rockchip1240-linux-gnu`
- **编译器**: `aarch64-rockchip1240-linux-gnu-gcc`

## 链接库
- **依赖库**: lvgl, pthread, m, rt, libdrm
- **头文件路径**: 包括项目源码及 `/sdk/media/libdrm/out/include` 等

## 配置文件
- `CMakeLists.txt` - 主构建配置 (已集成交叉编译工具链配置)
- `auto_build.sh` - 自动构建脚本 (编译失败将自动停止)
