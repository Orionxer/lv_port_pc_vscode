# 触摸功能配置说明

本文档简要说明如何在项目增加触摸屏支持。

## 1. 启用驱动支持

修改 `lv_conf.h` 文件，将 `LV_USE_EVDEV` 宏的值改为 `1`，以启用 Linux Evdev 输入驱动。

```c
// lv_conf.h

/** Driver for evdev input devices */
#define LV_USE_EVDEV    1
```

## 2. 添加初始化代码

修改 `src/main.c` 文件，在 `drm_hal_init()` 初始化之后，添加触摸设备的创建和绑定代码。

```c
// src/main.c

/*Initialize the HAL (display, input devices, tick) for LVGL*/
drm_hal_init();

/* Initialize touch input */
// 创建输入设备，指定类型为指针 (POINTER)，设备节点为 event1
lv_indev_t * indev = lv_evdev_create(LV_INDEV_TYPE_POINTER, "/dev/input/event1");

// 将输入设备绑定到默认显示器 (对于多屏系统或特定显示需求是必须的)
if(indev) {
    lv_indev_set_display(indev, lv_display_get_default());
}
```

> **注意**：设备路径 `/dev/input/event1` 是根据 `evtest` 测试结果（序号1）确定的。如果设备序号发生变化，请相应修改此路径。

## 3. 编译与运行

完成上述修改后，请重新运行构建脚本进行编译。

```bash
./auto_build.sh
```
